# WeatherIonic App

## Consume REST API

* import `HttpModule` from `@angular/http` to `app.module.ts`
* in the component that is going to make the call to the REST API import and inject `Http` component from `@angular/http`

```typescript
  getData() {
    // subscribe function register the handler or callback functions which will be executed
    // when http request either succeeds or fails
    this.http.get("http://jsonplaceholder.typicode.com/users").subscribe((data) => {
      console.log(data.json());
    }, (error) => {
      console.log(error);
    });
  }
```

      {{ user.name }}
